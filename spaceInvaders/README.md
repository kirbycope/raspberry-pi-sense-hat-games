# Space Invaders
An 8x8 pixel version of Space Invaders.</br>
![Alt text](/spaceInvaders/img/si_cover.png?raw=true "Cover art")

Octopus - Large (takes 3 hits)</br>

Crab - Medium (takes 2 hits)</br>
